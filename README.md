# Slack Homepage Clone

A fully responsive and visually appealing clone of the Slack homepage built using pure HTML and CSS. This project focuses on modern web design principles, featuring dynamic UI components, flexbox layouts, hover effects, and embedded video content.

## 🚀 Features

- **Modern UI Design:** Clean, pixel-perfect clone of Slack's landing page.
- **Dynamic Hero Section:** Interactive buttons and chips with hover states.
- **Embedded Videos & Media:** Integrates both images and auto-playing videos.
- **Custom Fonts & Icons:** Uses **Poppins** from Google Fonts and **Remix Icons** for scalable UI elements.
- **Card Layouts:** Beautifully styled "What's new" update cards with subtle hover animations and box shadows.

## 🛠️ Technologies Used

- **HTML5:** For semantic page structure.
- **CSS3:** For styling, Flexbox layouts, transitions, and hover effects.
- **Google Fonts:** Custom typography (Poppins).
- **Remix Icon:** For sleek, scalable vector icons.

## ⚙️ Setup and Installation

Running this project locally is extremely simple since it's a static frontend project.

### Method 1: Using VS Code Live Server (Recommended)
1. Open the project folder in **Visual Studio Code**.
2. Install the **Live Server** extension if you haven't already.
3. Open `index.html` in your editor.
4. Right-click anywhere in the code and select **Open with Live Server** (or click the "Go Live" button in the bottom right corner).
5. The project will automatically open in your default browser at `http://127.0.0.1:5500/`.

### Method 2: Using Node.js (npx)
If you have Node.js installed, you can quickly spin up a live server from your terminal:
1. Open your terminal and navigate to the project directory.
2. Run the following command:
   ```bash
   npx -y live-server
   ```
3. Your browser will automatically open and serve the files.

### Method 3: Direct File Execution
1. Navigate to the project folder on your computer.
2. Double-click on `index.html` to open it directly in your web browser. *(Note: Some features like auto-reloading won't work in this mode).*
